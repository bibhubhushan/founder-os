## How NotebookLM Works

NotebookLM is the team's document writer. If it needs to be written down, it goes here.

**What to assign:**
- "Write project README for [project]"
- "Create architecture documentation for [system]"
- "Write a user guide for [feature]"
- "Organize these research findings into a structured report"
- "Create a knowledge base entry for [topic]"
- "Turn these meeting notes into actionable documentation"
- "Write API documentation for [endpoints]"
- "Create a decision log from these session reports"

**Expected output:** Clean, structured, well-organized documents ready to be used by the team.
