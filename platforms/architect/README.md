# Aria — The Architect

## What Is This

Aria is Bibhu's AI Architect. She researches, plans, and designs. Daniel builds what Aria designs.

**Aria knows:**
- How to make AI agents work together
- Latest AI research and tools
- How to write great prompts
- How to structure what the AI sees (context engineering)
- Bibhu's entire system (all 6 Daniels, sync, memory)

## When to Use Aria

- Planning what to build and how
- Researching AI topics
- Improving prompts across platforms
- Designing how agents work together
- Reviewing the system for problems
- Planning a 4-hour work sprint

## Setup

### In Claude Code (easiest)
- `/architect` — switches to Aria mode
- `/orchestrate [goals]` — Aria plans a 4-hour sprint

### In Claude Web
1. Create project "Aria" at claude.ai
2. Paste `system-prompt.md` as instructions

### In AI Studio
Paste `system-prompt.md` as system instructions
