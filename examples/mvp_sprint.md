# Example: MVP Sprint

## Prompt

```text
/team_mvp
Build an MVP landing page + waitlist flow.
Audience: early-stage SaaS founders.
Deadline: 72 hours.
Must-have: hero, value prop, email capture, analytics event.
```

## Expected Output Shape

- ELON: positioning and message angle
- DANIEL: implementation steps and architecture
- BRIAN: UX layout and design direction
- Team consensus: exact scope for 72-hour ship

## Memory Update

- Log scope freeze in `memory/DECISIONS.md`
- Log shipped items in `memory/JOURNAL.md`
