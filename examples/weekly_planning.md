# Example: Weekly Planning (Founder)

## Prompt

```text
/coo
We need a weekly plan.
Goal: ship onboarding v1 by Friday.
Constraints: solo founder, 2 hours/day.
Done when: users can sign up, onboard, and see first value in < 3 min.
```

## Expected Output Shape

- One weekly metric
- Max 3 goals
- Daily tasks with owner/time estimate
- Friday review checklist

## Memory Update

- Add plan decision to `memory/DECISIONS.md`
- Add weekly objective to `memory/JOURNAL.md`
