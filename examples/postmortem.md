# Example: Weekly Postmortem

## Prompt

```text
/all_hands
We missed our weekly metric.
Expected: 30 signups.
Actual: 11 signups.
Need root causes and next-week plan.
```

## Expected Output Shape

- Root causes by domain (strategy, product, design, execution)
- 3 corrective actions
- One metric for next week

## Memory Update

- Add root-cause decision in `memory/DECISIONS.md`
- Add lessons in `memory/LESSONS.md`
- Append session eval in `memory/EVALS.md`
